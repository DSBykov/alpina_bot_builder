# alpina_bot_builder
Учебный проект от компании «Alpina Digital»: API конструктор ботов на основе GPT

## Установка и запуск

1. Клонировать репозиторий
2. Создать виртуальное окружение
3. Установить зависимости: `pip install -r requirements.txt`
4. Настроить переменные окружения в файле `.env` (`DJANGO_SECRET_KEY` и `GIGACHAT_AUTH_KEY`)
5. Выполнить миграции: `python manage.py migrate`
6. Создать суперпользователя: `python manage.py createsuperuser`
7. Запустить сервер: `python manage.py runserver` 

Этот проект предоставляет полнофункциональную систему для создания и управления ботами с поддержкой сценариев, интеграцией с GPT API и Telegram, а также удобным административным интерфейсом для управления всеми компонентами.
